# Amazon Review Sentiment Analysis

## Overview
This project performs sentiment analysis on Amazon product reviews using NLP techniques.

## Features
- Text preprocessing
- TF-IDF Vectorization
- Logistic Regression
- 88% Accuracy
- Streamlit Web Application

## Technologies
- Python
- Pandas
- NLTK
- Scikit-learn
- Streamlit

## Results
Achieved 88% accuracy on Amazon customer review data.

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
